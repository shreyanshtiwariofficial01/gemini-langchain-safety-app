
# 🚨 Gemini Model Using LangChain — Safety Settings Test (Streamlit App)

This project demonstrates how to use **Google Gemini** models through **LangChain** with customizable **Safety Settings**, all inside a simple and interactive **Streamlit** web application.

You can test how Gemini responds (or blocks) prompts based on different `safety_settings` thresholds such as:
- HARASSMENT  
- HATE_SPEECH  
- DANGEROUS_CONTENT  
- SEXUALLY_EXPLICIT  

---

## 📌 Features

- 🔹 Use **LangChain + Google Gemini** in a Streamlit UI  
- 🔹 Apply configurable **Safety Settings**  
- 🔹 Block or allow responses based on user input  
- 🔹 Clean and minimal interactive interface  
- 🔹 Environment variable support via `.env`
